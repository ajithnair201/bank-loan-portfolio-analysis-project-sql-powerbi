Bank Loan Portfolio Analysis

1.	Objective
-	A portion of a bank’s revenue, which is $28 million is locked in bad loans. Derive useful insights that can help the management to reduce bad loans % in their portfolio.

2.	Goal
-	Measure total loan applications, total amount disbursed, total revenue earned, average interest rate, average debt-to-income %.
-	Track good loan and bad loan percentage and the total loan applications, total amount disbursed and total revenue earned for each segment.
-	See loan status performance based on total loan applications, total amount disbursed, total revenue earned, average interest rate, average debt-to-income %.
-	Track monthly business growth
-	See proportion of loans disbursed based on home ownership, employment tenure, loan purpose and loan term.

3.	Analysis done
-	Created an SQL database with a table and pushed the data into it.
-	Did SQL data analysis of bank loan portfolio data to find total loans disbursed, total revenue earned, average interest rate and debt-to-income rate, Good and Bad loans issued, state wise loan distribution.
-	Connected Power BI Desktop with SQL database to pull the data into Power BI
-	Measured KPIs using DAX and Time intelligence functions.
-	Created charts for tracking good loan and bad loan composition.
-	Created a loan status table to see the portfolio performance.
-	Tracked monthly growth rate to see trends and patterns.

4.	Result
-	Built a interactive Power BI dashboard for strategic planning and decision-making to reduce bad loan percentage in the portfolio and improve profitability.
